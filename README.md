# BBTCAV
Calculating concept activation for Black Box Models

### Tests

#### Run AutoEncoder based BBTCAV on CelebA

In the main directory, run the following snippet:

`python -m tests.CelebA_AETCAV_test`

#### Run TCAV on CelebA

In the main directory, run the following snippet:

`python -m tests.CelebA_TCAV`
